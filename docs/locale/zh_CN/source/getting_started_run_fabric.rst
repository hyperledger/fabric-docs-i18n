============================

到目前为止，如果您已经下载了 Fabric 和相关样例，您可以开始运行 Fabric 。如果您还没有安装所需要的软件，请首先跟着下面的入门指导文档开始，请参考 :doc:`getting_started`

* 运行一个测试网络 - 帮助您理解 Fabric 网络运行原理。请参考教程 :doc:`Running a Test Network <test_network>` 
* 部署链码 - 帮助您开发和构建一个智能合约。请参考教程 :doc:`deploy_chaincode` 教程
* 手写第一个应用 - 帮助您在 Fabric 网络上开发区块链应用。请参考教程 :doc:`write_first_app` 教程

所有教程都有链接，想要更深入的理解，请参考 :doc:`key_concepts`
